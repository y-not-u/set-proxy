# set-proxy

a CLI to set proxy for npm, yarn, git or some other dev CLI more easily

## Usage

### Set Proxy

```bash
# set proxy for yarn
setproxy yarn http://127.0.0.1:7890
```

### Unset Proxy

```bash
setproxy -d yarn
```
